ES6 코드 실행

ES6 문법을 사용한 코드입니다. 구버전 브라우저에서는 실행되지 않을 수 있습니다. 


class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        return `안녕하세요, 저는 ${this.name}입니다. 나이는 ${this.age}살이에요.`;
    }
}

const person = new Person("홍길동", 25);
document.getElementById("output").innerText = person.greet();
