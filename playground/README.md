# 🎯 Babel Playground

Babel을 통해 최신 버전의 JavaScript 코드(ES6+)를 구형 브라우저에서도 동작 가능한 코드(ES5)로 변환하는 과정을 실시간으로 확인할 수 있습니다.

변환하고 싶은 코드를 직접 입력하고, Babel이 어떻게 코드를 변환하는지 바로 체험해보세요!

[🔗 바벨 플레이그라운드 바로가기](https://babel-playground-ebon.vercel.app/)
